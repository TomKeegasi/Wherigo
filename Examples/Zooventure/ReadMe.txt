The two lua files in this folder contain example code that demonstrates concepts learned in Levels Two and Three of the Wherigo Builder Tutorial. Users are encouraged to modify these files as needed.


Level Two concepts:
-- If-then-else statements
-- Standalone commands
-- Commands with one target
-- Commands with more than one target
-- Hiding commands

Level Three concepts:
-- Creating and incrementing variables
-- Setting up Inputs
-- Scripting Inputs


Level Three also contains two input-based puzzles. If you want to compile Level Three and play 
it on the emulator, you will need the following answers:

Puzzle 1 (how many volts does it take to turn the bathroom lights on?)
Answer: 1000

Puzzle 2 (What is the bird's name?)
Answer: sunbittern
